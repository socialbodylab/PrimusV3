# Overview
A slight adjustment to the hardware setup, I'm using the same esp32 module, but trying to connect to the 2 neopixel outputs using the featherwing via a adafruit doubler.
- Because I want the reverse TFT on the outside, i soldered the socket to the bottom of the doubler
- I also soldered the featherwing pins 'upside down' and then fit it to the socket on the back of the doubler

# Hardware
Same feather as before the esp32-s3 with the reverse tft
